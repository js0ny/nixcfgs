# nixcfgs

基于 [flake-parts](https://flake.parts/) 的 [NixOS](https://github.com/nixos/nixpkgs) 与 [nix-darwin](https://github.com/nix-darwin/nix-darwin) 配置文件

- [packages](https://github.com/js0ny/nix-misc-packages)
- [nvim](https://github.com/js0ny/nvimdots)

## 主机

- `bauhaus` - ITX(i5-13600KF + RTX4070Ti Super) ([包豪斯](https://en.wikipedia.org/wiki/Bauhaus))
- `crystal` - 幻14 Air 2025 ([水晶宫](https://en.wikipedia.org/wiki/The_Crystal_Palace))
- `polder` - 自托管 VPS ([圩田](https://en.wikipedia.org/wiki/Polder))
- `zen` - 运行 macOS 的 MacBook Air M1
- `zwinger` - Alt. VPS ([Zwinger](https://en.wikipedia.org/wiki/Zwinger))

## 我使用的软件

- 桌面会话
  - [Hyprland](https://hypr.land/) - 功能丰富
  - [Niri](https://github.com/niri-wm/niri) - 简单，低能耗
  - [KDE Plasma](https://kde.org/plasma-desktop/) - 备用环境
- 桌面外壳程序: [Noctalia V5](https://github.com/noctalia-dev/noctalia)
- 显示管理器: 使用 [thyx](https://github.com/rccyx/thyx) 主题的 [SDDM](https://github.com/sddm/sddm)
- 浏览器: [Firefox](https://www.firefox.com/en-GB/)
- 终端模拟器: [Kitty](https://sw.kovidgoyal.net/kitty/)
  - macOS: [Ghostty](https://ghostty.org/) 与 macOS 集成更好
- 命令行外壳程序: [fish](https://fishshell.com)
- 编辑器: [Neovim](https://neovim.io/)
- 终端复用器: [Kitty](https://sw.kovidgoyal.net/kitty/) 与 [zellij](https://zellij.dev/)
